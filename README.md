## ✦ 🚀 The Vision

Welcome to my personal **creative developer portfolio** repository. This project is a masterclass in combining the raw, unpolished aesthetics of **Brutalist Design** with the fluid, hyper-interactive motion capabilities of **GSAP (GreenSock Animation Platform)**. 

> *"It's not just a website; it's a high-fidelity digital experience."*

Built specifically to showcase advanced front-end engineering, this portfolio features complex velocity-based animations, magnetic physics, and an uncompromising, ultra-bold visual language.

## ✦ 🚀 Getting Started

Deploy this setup locally to explore the architecture and motion design.

<details>
<summary><b>Click to reveal installation instructions</b></summary>
<br/>

**1. Clone the repository**
```bash
git clone https://github.com/Anmol-Malviya/gsap-portfolio-nextjs.git
cd gsap-portfolio-nextjs
```

**2. Install dependencies**
```bash
npm install
```

**3. Fire up the development server**
```bash
npm run dev
```

**4. Experience it**
Navigate to `http://localhost:3000` to view the application in action.
</details>
